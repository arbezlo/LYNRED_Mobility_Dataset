# LYNRED_Mobility_Dataset
This repository contains the code to reproduce the figures in the paper : link_to_paper

## Datasets 

To reproduce the results given in the Figure 3 and Figure 5 of the paper, you will need to download the JSON files associated to each datasets. We compile the origin of the dataset's JSONs below : 

 * LLVIP : [link](https://drive.google.com/file/d/1VTlT3Y7e1h-Zsne4zahjx5q0TK2ClMVv/view?usp=sharing)
 * M3FD : [link](https://drive.google.com/drive/folders/1gvcf3jjjHnp8X8pKpwSDWhR7PHIuD6_J?usp=drive_link)
 * FLIR ADAS : [link](https://oem.flir.com/solutions/automotive/adas-dataset-form/)
 * KAIST : [link](https://drive.google.com/drive/folders/1lJLGV91CH43PRYx8KaVy5OngryTwo3ee?usp=drive_link)
 * LYNRED MD - Multispectral dataset : [link](https://www.lynred.com/fr/lynred-mobility-dataset)
 * FLIR Aligned : [link](https://drive.google.com/drive/folders/1-6iOnQmoNa9S8wwL7wYvqNH5IBcPSbj7?usp=drive_link)
